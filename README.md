# Advanced Hospital Management System (C Language)

A console-based Hospital Management System built in C using CSV file handling.  
This project implements real-world hospital operations such as patient records, doctor management, appointments, pharmacy inventory, and billing â€” all without requiring a database.

---

## Features

### Authentication
- Simple admin login (password: `govind123`)
- Prevents unauthorized access to system modules

### Patient Management
- Add new patients with full details
- Search patients by ID
- List all patients from CSV
- Delete patient records safely (temp file replace method)

### Doctor Management
- Maintain doctor profiles: name, specialization, and contact
- Search, list, and delete records
- Stored in `doctors.csv`

### Appointment Scheduling
- Create appointments between patient and doctor IDs
- Stores date and time in structured CSV format
- List and delete appointments

### Pharmacy / Medicine Inventory
- Add medicines with stock quantity and unit price
- View all available medicines
- Auto-update stock after billing

### Billing System
- Select medicines using `medID:qty` format
- Calculates line totals and grand total
- Automatically reduces stock in pharmacy
- Stores invoice in `bills.csv` with itemized details

### Reports Module
Quick access to:
- All patients  
- All doctors  
- All appointments  
- All medicines  
- All bills
